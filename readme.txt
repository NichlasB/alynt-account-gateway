A branded account gateway for WordPress login, registration, password flows, emails, dashboards, WooCommerce account handling, and integrations.

== Description ==

Alynt Account Gateway is a private Alynt-distributed plugin. The scaffold keeps frontend output disabled by default so account experiences can be configured and previewed before public behavior changes.
